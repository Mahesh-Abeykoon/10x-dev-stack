# Contributing

We love your input! We want to make contributing to this project as easy and transparent as possible.

## How to add a tools

1.  **Search** to see if the tool is already added.
2.  **Fork** the repo on GitHub.
3.  **Add** your tools in the appropriate category in alphabetical order.
    *   Format: `[Tool Name](URL) - Short description.`
4.  **Submit** a Pull Request.

## Guidelines

-   The tool must be relevant to AI coding, software development, or engineering.
-   Descriptions should be concise and objective.
-   Ensure links are working.
