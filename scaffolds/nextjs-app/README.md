# Next.js App Scaffold

A modern Next.js application scaffold optimized for AI-assisted development.

## Features

- Next.js 14 with App Router
- TypeScript
- ESLint configuration
- Optimized for AI code generation

## Getting Started

1. Copy this scaffold to your project
2. Run `npm install`
3. Run `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Why This Scaffold?

Pre-configured with settings that work well with AI tools like Cursor and Copilot.
