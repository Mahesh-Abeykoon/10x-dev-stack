# ⚛️ Vite + React (Optimized)

This is a clean, "no-fluff" scaffold for modern Web App development.

## Features
- **React 18**: The standard.
- **TailwindCSS**: Pre-configured.
- **TypeScript**: Strict mode enabled.
- **AI-Friendly**: The structure is flat and easy for Windsurf/Cursor to index.

## Usage

1. Copy this folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
