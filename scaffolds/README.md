These are "perfect starter" templates. Unlike standard `create-next-app` or `vite` templates, these are pre-configured with the **AI 10x Stack** in mind.

## Available Scaffolds

### 1. [Vite + React + TypeScript Optimized](./vite-react-optimized)
**Best for:** Web Applications, Dashboards, SaaS Frontend.
- **Why:** Pre-configured with Tailwind, specialized linting rules that don't fight AI, and a clean folder structure.

### 2. [FastAPI Agent Service](./fastapi-agent)
**Best for:** AI Agents, Backend Tools, Python APIs.
- **Why:** Structured for modular agents. Includes easy Docker setup and environment management.

### 3. [Minimal Local RAG (Python)](./minimal-rag-python)
**Best for:** Adding "Chat with PDF" to any project.
- **Why:** 50 lines of code. Uses Ollama + LanceDB. No heavyweight frameworks.

### 4. [MCP Server Starter](./mcp-server-starter)
**Best for:** Connecting Claude to your custom tools.
- **Why:** The "Hello World" of the Model Context Protocol. TypeScript based.

### 5. [Next.js App](./nextjs-app)
**Best for:** Full-stack web apps, SSR, API routes.
- **Why:** Modern Next.js 14 with App Router, TypeScript, and AI-optimized config.

### 6. [Express.js API](./express-api)
**Best for:** REST APIs, microservices, AI integrations.
- **Why:** Lightweight Express setup with security middleware and AI-ready endpoints.
